# Task 23 - Azure Resource Manager (ARM) & Cost Management

## 📌 Objective
To understand Azure Resource Manager (ARM), create Resource Groups, and analyze cost usage using Azure Cost Management.

This task demonstrates how Azure organizes resources and tracks cloud spending.

---

## 🛠️ Services Used
- Microsoft Azure Portal
- Azure Resource Manager (ARM)
- Resource Groups
- Azure Cost Management + Billing

---

## 🗂️ Part 1: Resource Group Creation (ARM)

### What is ARM?
Azure Resource Manager (ARM) is the deployment and management service in Azure.  
It allows grouping of resources for better organization, access control, and lifecycle management.

### Steps Performed:

1. Logged into Azure Portal.
2. Searched for **Resource Groups**.
3. Clicked **Create**.
4. Entered:
   - Subscription
   - Resource Group Name (e.g., RG-CloudLab)
   - Region
5. Clicked **Review + Create**.
6. Resource Group created successfully.

All Azure resources can now be deployed inside this group.

---

## 💰 Part 2: Cost Management & Analysis

### Steps Performed:

1. Opened **Cost Management + Billing**.
2. Navigated to **Cost Analysis**.
3. Selected:
   - Subscription
   - Time range (e.g., Current Month)
4. Viewed cost breakdown by:
   - Resource
   - Service
   - Resource Group
5. Observed usage trends and estimated cost.

Cost tracking dashboard displayed successfully.
