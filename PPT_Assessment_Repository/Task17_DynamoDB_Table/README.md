

#### \# Task 17: DynamoDB Table Creation (Basic)



\## 📌 Project Title



\*\*Creating and Managing a NoSQL Table using Amazon DynamoDB\*\*



---



\## 🎯 Objective



To create a NoSQL table using Amazon DynamoDB and insert sample items to understand serverless database fundamentals.



This task demonstrates understanding of:



\* NoSQL database concepts

\* Partition key usage

\* Serverless architecture

\* Item-based data storage



---



\## 🛠️ Services Used



\* Amazon DynamoDB

\* Amazon Web Services



---



\## 🏗️ Architecture Overview



1\. DynamoDB table is created.

2\. A primary key (Partition Key) is defined.

3\. Sample items are inserted.

4\. Data is retrieved from the table.



DynamoDB is fully managed and serverless — no infrastructure setup required.



---



\## ⚙️ Step-by-Step Implementation



---



\### Step 1: Create DynamoDB Table



1\. Go to \*\*DynamoDB Dashboard\*\*

2\. Click \*\*Create Table\*\*

3\. Keep default settings (On-demand capacity mode recommended)

4\. Click \*\*Create Table\*\*



Wait until status becomes \*\*Active\*\*



---



\### Step 2: Insert Sample Items



1\. Open the table → Click \*\*Explore table items\*\*

2\. Click \*\*Create Item\*\*

3\. Switch to \*\*JSON view\*\*

4\. Insert sample data:





Add another item



Click \*\*Create Item\*\*



---



\### Step 3: Retrieve Items



\* Go to \*\*Explore Items\*\*

\* Click \*\*Scan\*\*

\* View inserted records
